## 日期时间

- **date-fns**
  - 提供了最全面，最简单且一致的工具集，用于在浏览器和 Node.js 中操作 JavaScript 日期。
  - [官网](https://date-fns.org/)
  - [GitHub](https://github.com/date-fns/date-fns)
  - [npm](https://www.npmjs.com/package/date-fns)
- **Moment.js**
  - 一个 JavaScript 日期库，用于解析，验证，操作和格式化日期。
  - [官网](https://momentjs.com/)
  - [GitHub](https://github.com/moment/moment)
  - [npm](https://www.npmjs.com/package/moment)
- **Day.js**
  - 和 Moment.js 具有相同 API 的替代品，仅 2kB。
  - [官网](https://dayjs.gitee.io/zh-CN/)
  - [GitHub](https://github.com/iamkun/dayjs)
  - [npm](https://www.npmjs.com/package/dayjs)

## HTTP 相关

- **axios**
  - 基于 Promise 的 HTTP 客户端，用于浏览器和 node.js
  - [GitHub](https://github.com/axios/axios)
  - [npm](https://www.npmjs.com/package/axios)

- **js-cookie**
  - 一个简单，轻巧的 JavaScript API，用于处理 Cookie
  - [GitHub](https://github.com/js-cookie/js-cookie)
  - [npm](https://www.npmjs.com/package/js-cookie)

## 图像处理

- **DOM to Image**
  - 将任意 DOM 节点转换为用 JavaScript 编写的矢量（SVG）或光栅（PNG 或 JPEG）图像。
  - [GitHub](https://github.com/tsayen/dom-to-image)
  - [npm](https://www.npmjs.com/package/dom-to-image)

## 数据可视化

- **Apache ECharts**
  - 一个免费的，功能强大的图表和可视化库（不争气的百度卖给 Apache 了）
  - [官网](https://echarts.apache.org/zh/index.html)
  - [GitHub](https://github.com/apache/echarts)
  - [npm](https://www.npmjs.com/package/echarts)

## 拖拽

- **dnd.js**
  - 不依赖任何第三方库的拖放库，兼容低版本浏览器，兼容移动端，自带常用动画效果。
  - [npm](https://www.npmjs.com/package/dnd.js)
- **Droppable**
  - 一个轻量级的，响应式的现代 JavaScript 拖放库。
  - [官网](https://shopify.github.io/draggable/examples/)
  - [GitHub](https://github.com/Shopify/draggable)
  - [npm](https://www.npmjs.com/package/@shopify/draggable)

## 编辑器

- **TinyMCE**
  - 功能强大、所见即所得的富文本编辑器
  - [中文文档](http://tinymce.ax-z.cn/)
  - [官网](https://www.tiny.cloud/docs/quick-start/)
  - [GitHub](https://github.com/tinymce/tinymce)
  - [npm](https://www.npmjs.com/package/tinymce)
- **wangEditor**
  - Typescript 开发的 Web 富文本编辑器， 轻量、简洁、易用、开源免费
  - [官网](https://www.wangeditor.com/)
  - [GitHub](https://github.com/wangeditor-team/wangEditor/)
  - [npm](https://www.npmjs.com/package/wangeditor)
- **JSON Editor**
  - 基于 Web 的工具，用于查看、编辑、格式化和验证 JSON。它具有多种模式，例如树编辑器、代码编辑器和纯文本编辑器。
  - [GitHub](https://github.com/josdejong/jsoneditor)
  - [npm](https://www.npmjs.com/package/jsoneditor)
- **CodeMirror**
  - JavaScript 实现的通用文本编辑器。它专门用于编辑代码，并具有 100 多种语言模式和各种插件，可实现更高级的编辑功能。每种语言都带有功能齐全的代码和语法突出显示，以帮助阅读和编辑复杂的代码。
  - [官网](https://codemirror.net/)
  - [GitHub](https://github.com/josdejong/jsoneditor)
  - [npm](https://www.npmjs.com/package/jsoneditor)

## 其他实用插件

- **NProgress**
  - 适用于 Ajax'y 应用程序的超薄进度条。受到 Google，YouTube 和 Medium 的启发。
  - [GitHub](https://github.com/rstacruz/nprogress)
  - [npm](https://www.npmjs.com/package/nprogress)
- **uuid**
  - 用于创建 [RFC4122](http://www.ietf.org/rfc/rfc4122.txt) UUID
  - [GitHub](https://github.com/uuidjs/uuid)
  - [npm](https://www.npmjs.com/package/uuid)
- **clipboard.js**
  - 复制到剪贴板。不需要 Flash。压缩后只有 3kb。
  - [官网](https://clipboardjs.com/)
  - [GitHub](https://github.com/zenorocha/clipboard.js)
  - [npm](https://www.npmjs.com/package/clipboard)
- **SheetJS js-xlsx**
  - 用于解析和编写各种格式的 Excel。
  - [GitHub](https://github.com/SheetJS/sheetjs)
  - [npm](https://www.npmjs.com/package/xlsx)
- **crypto-js**
  - 用于加密解密的 JavaScript 库。
  - [GitHub](https://github.com/brix/crypto-js)
  - [npm](https://www.npmjs.com/package/crypto-js)
- **FileSaver.js**
  - FileSaver.js 是在客户端保存文件的解决方案，非常适合在客户端上生成文件的 Web 应用程序。
  - [GitHub](https://github.com/eligrey/FileSaver.js/)
  - [npm](https://www.npmjs.com/package/file-saver)
